Os valores possíveis:

  • ‘name’ – Nome do site. Valor padrão.
  • ‘description’ – Descrição do site.
  • ‘url’ – O endereço do site.
  • ‘rdf_url’ – Endereço do RDF/RSS 1.0.
  • ‘rss_url’ – Endereço do RSS 0.92.
  • ‘rss2_url’ – Endereço para RSS 2.0.
  • ‘atom_url’ – URL Feed Atom.
  • ‘comments_rss2_url’ – Url para acesso do RSS 2.0 dos comentários.
  • ‘pingback_url’ – Url dos Pingbacks Pingback (XML-RPC file).
  • ‘admin_email’ – Endereço de email do administrador do WordPress.
  • ‘charset’ – Tabela de caractéres utilizada no Worpdress; Você define isso na Opções de Leitura no Admin.
  • ‘version’ – Mostra a versão do WordPress que você utiliza..
  • ‘text_direction’ – Retorna ‘rtl’ leitura da direita para esquerda ou ‘ltr’ para leitura da esquerda para direita, que é o padrão.
