Você já usou aquela opção da Web Developer de editar o CSS e ver o resultado no Firefox na hora? E quando você usava, você não pensou: – Po, isso podia funcionar de alguma forma no IE…

Seu problemas acabaram. O CSSVista é uma aplicação Windows que permite você editar o CSS do seu site e ver o resultado “ao vivo” simultanemente no Firefox e no Internet Explorer.

Como nada é perfeito, para poder funcionar, você deve instalar o .NET Framework do Windows.

Por causa desse inconveniente, eu não instalei, por isso não posso dizer como é o software.

Mesmo assim, uma bela iniciativa.
