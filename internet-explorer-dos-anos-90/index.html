Internet Explorer dos anos 90

You grew up. So did we. Reconnect with the new Internet Explorer.

por Diego Eis

A Microsoft liberou mais um comercial muito, mas muito bom do Internet Explorer.

Dessa vez ela resolveu fazer uma homenagem a todos que viveram nos anos 90, mostrando todas as “coisinhas” que viraram moda naquela época, tipo ioiô, tamagochi e até o tazo.

You grew up. So did we. Reconnect with the new Internet Explorer.
